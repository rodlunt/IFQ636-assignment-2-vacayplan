## Files

| File | Purpose |
|------|---------|
| `A2_Brief.md` | Assignment summary, Part A/B/C structure, verbatim template-section map |
| `A2_Rubric.md` | Canonical rubric (verbatim) + full grade-band descriptors + design-pattern matrix |
| `A2_Plan.md` | Phase-based delivery plan + critical-path dependencies + risk register |
| `A2_Checklist.md` | Rubric checklist, pattern tracker, screenshot tracker, submission + team-coordination checklists |
| `A2_Report_Notes.md` | **Live** running notes organised by exact template headings - source for write-up day |
| `screenshots/` | Captured screenshots (created as work progresses) |

## Workflow

1. Work through `A2_Plan.md` phases (Phase 0 is solo-actionable now)
2. Capture decisions/facts/references/GenAI uses into `A2_Report_Notes.md` live, never reconstruct at the end
3. Tick boxes + fill trackers in `A2_Checklist.md` as evidence is captured
4. Drop screenshots into `screenshots/` immediately after capture (full screen + avatar visible)
5. Session-end: log against `IFQ636_Study_Plan.md` Session Log
